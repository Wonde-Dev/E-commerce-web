# 🛒 E-Commerce Website (React + Tailwind CSS)

A modern, responsive eCommerce web application built using **React** and **Tailwind CSS**. This project provides a clean UI and smooth shopping experience.

---

## 🚀 Features

- 🛍️ Product listing and details page  
- 🔍 Search and filter products  
- 🛒 Add to cart functionality  
- 💳 Checkout UI design  
- 📱 Fully responsive layout  
- ⚡ Fast performance with Vite  
- 🎨 Beautiful UI with Tailwind CSS  

---

## 🛠️ Tech Stack

- React JS  
- Tailwind CSS  
- Vite  
- JavaScript (ES6+)
